# Electric_Vehicles_Data_Analysis_Project
Analyzed Electric Vehicles Sales and Usage using Tableau and MS-Excel

📊 Electric Vehicles Data Analysis (Tableau Dashboard)
🚀 Project Overview

This project presents an interactive Electric Vehicles (EV) Data Analysis Dashboard built using Tableau.
The goal is to analyze EV adoption trends, identify key patterns, and generate actionable insights related to vehicle growth, geography, manufacturers, and policy impact.

🎯 Objectives
Analyze EV adoption trends over time

Identify top-performing states and regions

Compare BEV vs PHEV distribution

Evaluate manufacturer and model performance
Understand the impact of CAFV eligibility

📁 Dataset Description

The dataset contains information on:

🚗 Vehicle Type (BEV / PHEV)
🏭 Manufacturer (Make & Model)
📅 Model Year
🌍 State-wise distribution
🔋 Electric Range
📊 CAFV (Clean Alternative Fuel Vehicle) Eligibility
📊 Dashboard Features
🔑 Key Performance Indicators (KPIs)
Total Vehicles: 150,437
Average Electric Range: 67.86 km
BEV Share: 77.6%
PHEV Share: 22.4%
📈 Visualizations Included
EV Growth Trend (Year-wise Analysis)
State-wise EV Distribution (Map)
Top 10 Manufacturers (Bar Chart)
CAFV Eligibility (Donut Chart)
Model-wise Vehicle Distribution (Table)
🔍 Key Insights
📈 Growth Trend
EV adoption shows exponential growth after 2018
Peak registrations observed in 2023
Indicates transition toward mass adoption phase
🌍 Geographic Insights
California dominates EV adoption (~92K vehicles)
EV usage is concentrated in a few regions
Highlights infrastructure dependency
🚗 Vehicle Type Insights
BEVs dominate (77.6%) over PHEVs (22.4%)
Shows strong shift toward fully electric vehicles
🏭 Manufacturer Insights
Tesla leads with ~52.7% market share
Followed by Nissan and Chevrolet
Indicates high market concentration
🚘 Model Insights
Top models:
Tesla Model Y
Tesla Model 3
Nissan Leaf
Few models contribute to majority of adoption
📊 CAFV Eligibility Insights
Eligible: 41.82%
Not Eligible: 11.85%
Unknown: 46.33%

⚠️ Indicates data quality issues and need for better tracking

# 📊 Electric Vehicles Data Analysis (Tableau)

## 🚀 Project Overview

This project showcases an interactive **Electric Vehicles (EV) Data Analysis Dashboard** built using Tableau. The objective is to analyze EV adoption trends, uncover patterns, and generate meaningful insights for business decision-making.

---

## 🎯 Objectives

* Analyze EV growth trends over time
* Identify top-performing states
* Compare BEV vs PHEV adoption
* Evaluate manufacturer and model performance
* Understand CAFV eligibility impact

---

## 📁 Dataset Details

The dataset includes:

* 🚗 Vehicle Type (BEV / PHEV)
* 🏭 Make & Model
* 📅 Model Year
* 🌍 State
* 🔋 Electric Range
* 📊 CAFV Eligibility

---

## 📊 Dashboard Features

### 🔑 Key KPIs

* **Total Vehicles:** 150,437
* **Average Electric Range:** 67.86 km
* **BEV Share:** 77.6%
* **PHEV Share:** 22.4%

---

### 📈 Visualizations

* EV Growth Trend (Year-wise)
* State-wise EV Distribution (Map)
* Top 10 Manufacturers (Bar Chart)
* CAFV Eligibility (Donut Chart)
* Model-wise Distribution (Table)

---

## 🔍 Key Insights

### 📈 Growth Trend

* Rapid increase after 2018
* Peak in 2023
* Indicates **exponential growth phase**

---

### 🌍 Geographic Insights

* California leads EV adoption (~92K vehicles)
* Adoption concentrated in few regions
* Shows **infrastructure dependency**

---

### 🚗 Vehicle Type

* BEVs dominate (77.6%)
* Strong shift toward **fully electric vehicles**

---

### 🏭 Manufacturer Analysis

* Tesla leads with ~52.7% share
* Followed by Nissan & Chevrolet
* Indicates **high market concentration**

---

### 🚘 Model Insights

* Top models:

  * Tesla Model Y
  * Tesla Model 3
  * Nissan Leaf
* Few models dominate overall share

---

### 📊 CAFV Eligibility

* Eligible: 41.82%
* Not Eligible: 11.85%
* Unknown: 46.33%

⚠️ Data quality improvement needed
